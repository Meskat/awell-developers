---
title: Get published pathways
description: Get all published pathways in your organization
---

Gets the full list of published pathway definitions in your tenant or organization.

## Request

### Query

```graphql
query publishedPathwayDefinitions {
  publishedPathwayDefinitions {
    publishedPathwayDefinitions {
      id
      title
      dataPointDefinitions {
        id
        key
        valueType
        optional
      }
    }
  }
}
```
