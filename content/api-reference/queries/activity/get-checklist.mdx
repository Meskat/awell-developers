This request allows us to fetch the details of a checklist activity.

## Request

### Query

```graphql
query GetChecklist($id: String!) {
  checklist(id: $id) {
    checklist {
      title
      items
    }
  }
}
```

### Variables

```json
{
  "id": "{{CHECKLIST_ID}}"
}
```

This query expects only one input variable:
