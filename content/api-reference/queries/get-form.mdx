---
title: Get form
description: Get the form definition of a given form
---

This query allows us to fetch the details and questions for a specific form.

## Request

### Query

```graphql
query GetForm($id: String!) {
  form(id: $id) {
    form {
      id
      title
      questions {
        id
        title
        dataPointValueType
        questionType
        options {
          id
          value
          label
        }
      }
    }
  }
}
```

### Variables

An example `id` for a form is "9rjfb9qE4pEY".

```json
{
  "id": "{{FORM_ID}}"
}
```
