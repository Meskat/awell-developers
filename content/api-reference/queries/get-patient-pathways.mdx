---
title: Get patient pathways
description: Get all pathways a given patient is or was enrolled in
---

## Request

### Query

```graphql
query GetPatientPathways($patient_id: String!) {
  patientPathways(patient_id: $patient_id) {
    patientPathways {
      id
      title
      pathway_definition_id
      status
      status_explanation
      version # version number
      release_id # unique id representing the version
      start_date
      stop_date # if pathway is stopped
      complete_date # if pathway is completed
      baseline_info {
        value
        definition {
          id
          title
          category
          key
          valueType
          possibleValues {
            label
            value
          }
          unit
          range {
            min
            max
          }
        }
      }
    }
  }
}
```

### Variables

```json
{
  "patient_id": "{{PATIENT_ID}}"
}
```
