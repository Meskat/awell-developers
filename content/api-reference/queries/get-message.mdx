---
title: Get message
description: Get the body and attachments of a message activity
---

This request allows us to fetch the details of a message activity.

## Request

### Query

```graphql
query GetMessage($id: String!) {
  message(id: $id) {
    message {
      id
      body
      format
      subject
      attachments {
        id
        name
        url
        type
      }
    }
  }
}
```

### Variables

```json
{
  "id": "{{MESSAGE_ID}}"
}
```

This mutation expects only one input variable:
