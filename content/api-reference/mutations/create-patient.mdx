---
title: Create patient
description: Create a new patient, anonymous patients are also supported.
---

## Request

### Query

```graphql
mutation createPatient($input: CreatePatientInput!) {
  createPatient(input: $input) {
    patient {
      id
    }
  }
}
```

### Variables

```json
{
  "input": {
    "first_name": "{{PATIENT FIRST NAME}}",
    "last_name": "{{PATIENT LAST NAME}}",
    "patient_code": "{{PATIENT CODE}}",
    ...
  }
}
```

## Support for anonymous patients

As all of the input fields are optional, this means, in effect, that you can create anonymous patients should you need to. For example, you can begin a patient journey with no details provided except an internal patient_code reference and use the generated patient ID to progress them through a pathway until such time you wish to gather and update their personal details.
