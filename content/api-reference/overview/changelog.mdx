Please note that this page does not list all releases and all changes in a release but rather only those releases and changes that have an impact on the consumers of our API.

## 1.15.2.2 (June 1, 2022)

**startPathway mutation:**

- Addressed a performance issue with the mutation when calling it for a pathway that has Webhooks configured

## 1.15.1 (May 30, 2022)

**getScheduledSteps query:**

- New query to fetch the scheduled (i.e. future) steps in a given pathway

**pathwayDataPointDefinitions query**

- Fix issue caused by filters set to null that resulted in an error

## 1.14.2.3 (May 27, 2022)

**searchPatientByPatientCode query:**

- Fix destruction issue causing the following error: "Cannot read properties of undefined (reading 'national_registry_number')" introduced by a breaking change in v0.14.2

## 1.14.2 (May 9, 2022)

**Generic:**

- Improved error messages

**activities query:**

- Ability to paginate & sort the response

**patients query:**

- Ability to filter on the following fields: `national_registry_number`, `patient_code`, and `profile_id`
- Ability to paginate & sort the response

**pathway query:**

- Additional fields available in response: `version`, `release_id`, `stop_date`, `complete_date`, `status_explanation` (eg: reason why pathway was stopped)

**pathways query:**

- New query to search pathways by status, type, patient and other criteria (incl. filters, pagination, and sorting)

**patientsPathways query:**

- Ability to filter by pathway status
- Baseline info is now available in the response (`baseline_info` field)

**pathwayDataPointDefinitions query**

- New query to fetch the data catalog for a given version of a pathway definition

**publishedPathwayDefinitions query:**

- Additional fields available in response: `version`

## 1.14.1 (May 2, 2022)

**activities query:**

- New query with the ability to filter so you can get the activities you need

**patientsPathways query:**

- Additional fields available in response: `version`, `release_id`, `stop_date`, `complete_date`, `status_explanation` (eg: reason why pathway was stopped)

## Earlier versions (before May 2022)

We started publishing a public changelog since May 2022. Send us as message via Intercom if you need more information on earlier versions. We're always happy to provide you more information.
