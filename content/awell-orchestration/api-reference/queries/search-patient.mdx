---
title: Search patients
description: Search patients on patient code or national registry number
---

Search for one or more patients in your tenant by patient code or national registry number, regardless of pathway or patient status. The search is done via partial-text matching so multiple results may be returned for any given query.

### Query

Search by patient code:

```graphql
query SearchPatientsByPatientCode($patient_code: String!) {
  searchPatientsByPatientCode(patient_code: $patient_code) {
    patients {
      id
    }
  }
}
```

Search by national regsitry number:

```graphql
query SearchPatientsByNationalRegistryNumber(
  $national_registry_number: String!
) {
  searchPatientsByNationalRegistryNumber(
    national_registry_number: $national_registry_number
  ) {
    patients {
      id
    }
  }
}
```

### Variables

Search by patient code:

```json
{
  "patient_code": "{{PATIENT_CODE}}"
}
```

Search by national regsitry number:

```json
{
  "national_registry_number": "{{NATIONAL_REGISTRY_NUMBER}}"
}
```
